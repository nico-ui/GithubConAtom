Introducción al desarrollo web: HTML y CSS

La web es un sistema a través del cual se gestiona información que será compartida por Internet. Es necesario tener una conexión a Internet en el ordenador o dispositivo móvil para poder consultar cualquier cosa en dicha web a través de un navegador como por ejemplo Chrome, Mozilla, Safari, etc. Todos emplean el sistema de Localizador Uniforme de Recursos o URL, que es la dirección concreta que define a cada uno de los recursos que se encuentran en la red. Es decir, es un identificador propio para cada página, documento, archivo, entre otros, que se encuentre en la web.
